# Vending Machine App
The Vending Machine App is a console-based application that simulates the
basic functionality of a real vending machine.

## Products
The vending machine sells different types of products, such as:
* Snacks
* Beverage
* Fruits

## Features
The vending machine allows a user to:
* View available products
* Select a product
* Purchase a product
* Reduce stock after purchase
* Interact using a console-based menu

## Accepted Coins
The vending machine accepts only specific coin values: 1, 2, 5, 10, 20, 50

## Rules:
* Any other coin value must be rejected.
* Inserted coin increase the current balance.
* A product can only be purchased if the balance is sufficient.
* Remaining balance must be returned when requested.
